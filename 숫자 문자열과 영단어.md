## 문제

네오와 프로도가 숫자놀이를 하고 있습니다. 
네오가 프로도에게 숫자를 건넬 때 일부 자릿수를 영단어로 바꾼 카드를 건네주면 프로도는 원래 숫자를 찾는 게임입니다.
다음은 숫자의 일부 자릿수를 영단어로 바꾸는 예시입니다.
- 1478 → "one4seveneight"
- 234567 → "23four5six7"
- 10203 → "1zerotwozero3"

## 코드 

### 나의 코드
while의 조건문을 이용해 반복을 주고 replace() 메서드를 사용해 문자를 숫자로 바꿔줌.
```
const s = "one4oneoneone";
const reg = /one/;
let repair=s;
while(repair.includes("one")){repair= repair.replace("one","1");}
while(repair.includes("two")){repair= repair.replace("two","2");}
while(repair.includes("three")){repair= repair.replace("three","3");}
while(repair.includes("four")){repair= repair.replace("four","4");}
while(repair.includes("five")){repair= repair.replace("five","5");}
while(repair.includes("six")){repair= repair.replace("six","6");}
while(repair.includes("seven")){repair= repair.replace("seven","7");}
while(repair.includes("eight")){repair= repair.replace("eight","8");}
while(repair.includes("nine")){repair= repair.replace("nine","9");}
while(repair.includes("zero")){repair= repair.replace("zero","0");}
Number(repair);
console.log(repair)
```

### 생각하지 못했던 풀이 (by 대균님)
배열로 one~nine 까지 선언. s의 길이가 최대 50 이므로 for 문을 통해 50번 돌려준다.
(숫자가 껴있을 수 있으므로 i를 48~50 정도로 돌려주는게 안전)
if문의 조건을 통해 문자를 포함하고있는지 확인 후
배열의 index와 i%10 이 같으므로 replace함수를 이용해 바꿔주면 된다.
```
function solution(s) {
    var answer = 0;
    let arr = ["zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"]
    for (let i =0; i<= 50; i++){
        if(s.includes(arr[i%10])){
        s = s.replace(arr[i%10], i%10)
        }
    }
    answer = parseInt(s)
    return answer;
}
```

### Best 풀이 (by 윤님)
배열로 one~nine 까지 만들어 준뒤 for 문을 통해 split을 통해 단어별로 구분해준다.
join을 통해 각각 에 해당하는 숫자를 넣어준뒤 문자열로 바꿔준다.

```
function solution(s) {
    let numbers = ["zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"];
    var answer = s;

    for(let i=0; i< numbers.length; i++) {
        let arr = answer.split(numbers[i]);
        answer = arr.join(i);
    }

    return Number(answer);
}
```
