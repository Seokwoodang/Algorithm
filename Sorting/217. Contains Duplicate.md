## [217. Contains Duplicate](https://leetcode.com/problems/contains-duplicate/description/?envType=problem-list-v2&envId=sorting)

- 문제출처 : leetcode
- 개념 : sorting
- 난이도 : easy


```
var containsDuplicate = function(nums) {
    let map = new Map();
    for(let x of nums){
        if(map.has(x)){
            return true
        }else{
            map.set(x,1)
        }
    }
    return false
};
```
