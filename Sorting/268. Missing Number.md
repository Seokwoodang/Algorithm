## [268. Missing Number](https://leetcode.com/problems/missing-number/description/?envType=problem-list-v2&envId=sorting)

- 문제출처 : leetcode
- 개념 : sorting
- 난이도 : easy



```
var missingNumber = function(nums) {
    let total = 0;
    for(let i = 0 ; i<=nums.length;i++){
        total += i;
        if(i<nums.length){
            total -= nums[i]
        }
    }
    return total
};
```
