## [349. Intersection of Two Arrays](https://leetcode.com/problems/intersection-of-two-arrays/description/?envType=problem-list-v2&envId=sorting)

- 문제출처 : leetcode
- 개념 : sorting
- 난이도 : easy


```
var intersection = function(nums1, nums2) {
    let map = new Map();
    let answer = []
    for(let x of nums1){
        map.set(x,1);
    }
    for(let x of nums2){
        if(map.has(x)){
            answer.push(x);
            map.delete(x);
        }
    }
    return answer
};
```
