## [71. Simplify Path](https://leetcode.com/problems/simplify-path/description/?envType=problem-list-v2&envId=stack&difficulty=MEDIUM)

- 문제출처 : leetcode
- 개념 : 스택
- 난이도 : medium


```
var simplifyPath = function(path) {
    const stack = []
    const arr = path.split('/').filter((item)=>item!=='')
    for(const x of arr){
        if(x==='..'){
            stack.pop();
        }else if(x!=='.'){
            stack.push(x)
        }
    }
    return '/'+stack.join('/');
};
```
